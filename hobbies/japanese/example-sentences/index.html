Questions in Japanese : か particle

  1. それは上手いですか？
  2. 私は人ですか？
  3. 今日は暑いですか？

Using の to add Information

  1. それは闇の刀ですよ！　～　Describes aspect
  2. いや！それはアッシャーさんのコンピューターですよ！　～　Describes ownership
  3. これは何のためですか？　～　What but as an aspect of thing

  1. これは可愛いそう
  2. それは何ですか？

この, その, etc

  1. この猫は黒い
  2. そのアパートは狭い
  3. あの温泉は暑い
  4. どの色好き？

ここ, そこ, etc

  1. 式はここですか？
  2. なおさんー　そこは何ですか？
  3. あー、コンビニはあそこです
  4. どこに行こうか？
